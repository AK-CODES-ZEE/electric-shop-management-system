# Electric Shop Management System (Oracle APEX)

## Project Overview

The Electric Shop Management System is a real-world business application 
developed using Oracle APEX. The system is designed to manage products, 
inventory, customers, and sales operations for an electrical retail shop.

This project demonstrates practical usage of Oracle APEX, SQL, and PL/SQL 
to build a secure, scalable, and user-friendly enterprise application.

---

## Technologies Used

- Oracle APEX
- Oracle Database
- SQL
- PL/SQL
- API-based AI Integration (Basic Configuration Demo)

---

## Key Features

### 1. Product Management
- Add, update, and delete electrical products
- Track product categories and pricing
- Maintain stock quantities

### 2. Inventory Management
- Real-time inventory tracking
- Low stock monitoring
- Automated stock updates after sales

### 3. Sales Management
- Sales entry form
- Customer selection and billing
- Sales reports with filtering options

### 4. Dashboard & Reports
- Sales summary dashboard
- Inventory overview
- Product-wise and date-wise reports

### 5. AI Integration (Configuration Demo)
- Basic AI configuration using API-based integration
- Demonstration of intelligent suggestions / automation logic
- AI-assisted data insights within the application (concept-level demo)

---

## Application Architecture

The application follows a structured architecture:

- Frontend: Oracle APEX Pages (Forms, Interactive Reports, Dashboards)
- Backend: Oracle Database
- Business Logic: PL/SQL Procedures and Triggers
- AI Integration: External API configuration within APEX
